Detailed Description

template<unsigned int Dimension>
class featkAttributable< Dimension >

Base class for featk objects possessing attributes, i.e. featkNode and featkElementInterface.

featkAttributable is a base class for featk objects with attributes (featkNode and featkElementInterface), providing member variables and functions for attribute assignment, storage and access.

Attributes are stored as std::shared_ptr<Eigen::MatrixXd>. std::shared_ptr allows featkAttributable objects to share attributes without having to duplicate them nor to worry about memory mangement. Dynamic-sized Eigen::MatrixXd allows to store attributes of various Order seamlessly.
