\doxysubsection{Detailed Description}
Node with its attributes including coordinates in Dimension cartesian dimensions. 

\mbox{\hyperlink{classfeatk_node}{featk\+Node}} is the implementation of a node with its attributes including coordinates in Dimension cartesian dimensions.

\mbox{\hyperlink{classfeatk_node}{featk\+Node}} stores a pointer to each \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}} it belongs to for computational efficiency when evaluating node derivative quantities that need to be averaged over elements sharing this node.

Each \mbox{\hyperlink{classfeatk_node}{featk\+Node}} must be given a unique id at construction time whose uniqueness is ensured by the user.

\begin{DoxyWarning}{Warning}
\mbox{\hyperlink{classfeatk_node}{featk\+Node}} cartesian coordinates are automatically registered as a node attribute in the \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} constructor under name \char`\"{}\+Cartesian
\+Coordinates\char`\"{}. Make sure not to assign node attribute of order 1 with name \char`\"{}\+Cartesian Coordinates\char`\"{} to \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} objects except if you intend to modify the \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} geometry.
\end{DoxyWarning}
