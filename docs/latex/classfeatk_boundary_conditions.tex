\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int Dimension, unsigned int Order$>$\newline
class featk\+Boundary\+Conditions$<$ Dimension, Order $>$}

Boundary conditions for finite element problems. 

\mbox{\hyperlink{classfeatk_boundary_conditions}{featk\+Boundary\+Conditions}} stores the ids of the fixed degrees of freedom of finite element problems along with their prescribed values.

For efficiency reasons during the assembly process (see \mbox{\hyperlink{classfeatk_solver_base}{featk\+Solver\+Base}}), distinction is made between zero prescrided values, for which the global system vector can be trivially modified, and non-\/zero prescrided values, involving iteration over non-\/zeros entries of the global system matrix.


\begin{DoxyTemplParams}{Template Parameters}
{\em Dimension} & The cartesian dimension of the problem.\\
\hline
{\em Order} & The order of the variable the system is solved for. \\
\hline
\end{DoxyTemplParams}
