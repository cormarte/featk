Base class for featk objects possessing attributes, i.\+e. \mbox{\hyperlink{classfeatk_node}{featk\+Node}} and \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}}. 

\mbox{\hyperlink{classfeatk_attributable}{featk\+Attributable}} is a base class for featk objects with attributes (\mbox{\hyperlink{classfeatk_node}{featk\+Node}} and \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}}), providing member variables and functions for attribute assignment, storage and access.

Attributes are stored as std\+::shared\+\_\+ptr$<$\+Eigen\+::\+Matrix\+Xd$>$. std\+::shared\+\_\+ptr allows \mbox{\hyperlink{classfeatk_attributable}{featk\+Attributable}} objects to share attributes without having to duplicate them nor to worry about memory mangement. Dynamic-\/sized Eigen\+::\+Matrix\+Xd allows to store attributes of various Order seamlessly.
