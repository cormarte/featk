\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int Dimension, unsigned int Order$>$\newline
class featk\+Dynamic\+Solver\+Base$<$ Dimension, Order $>$}

Base class for dynamic finite element solvers. 

\mbox{\hyperlink{classfeatk_dynamic_solver_base}{featk\+Dynamic\+Solver\+Base}} is a base class for dynamic finite element solvers. \mbox{\hyperlink{classfeatk_dynamic_solver_base}{featk\+Dynamic\+Solver\+Base}} introduces the concept of iterative time-\/step solving and provides convenient mechanisms for stepwise global system vector recalculation and intermediate solution processing.

Typical iterative solving is perfomed as follows\+:

1) Global matrices involved that are not subject to change throughout the solving process are computed inside the overriden virtual member function \mbox{\hyperlink{classfeatk_solver_base_a0041283d0bbb41b0955e511469ce3b65}{featk\+Solver\+Base\+::initialize()}}.

2) The global system matrix is generated from the global matrices assembled in 1) and E\+B\+Cs are applied to it.

3) An initial solution vector is generated.

4) For each iteration\+:~\newline
a. A global system vector is generated from the global matrices computed in 1) and the previous solution.~\newline
b. E\+B\+Cs are applied to the global vector.~\newline
c. System is solved.~\newline
d. Current solution is cut off if required.~\newline
e. Intermediate processing is performed on the current solution if required.
