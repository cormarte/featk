\doxysubsection{Detailed Description}
Interface for \mbox{\hyperlink{classfeatk_element}{featk\+Element}} objects in Dimension cartesian dimensions. 

\mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}} is an interface allowing to handle \mbox{\hyperlink{classfeatk_element}{featk\+Element}} objects with Dimension cartesian dimensions and various Nodes, Boundaries and Natural\+Dimension template parameter values. \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}} makes it possible to define hybrid \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} seamlessly.
