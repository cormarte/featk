\doxysubsection{Detailed Description}
Mesh in Dimension dimensions with its elements and nodes. 

\mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} gather \mbox{\hyperlink{classfeatk_element}{featk\+Element}} and \mbox{\hyperlink{classfeatk_node}{featk\+Node}} objects defining the geometry of a finite element problem.

\mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} manages \mbox{\hyperlink{classfeatk_element}{featk\+Element}} and \mbox{\hyperlink{classfeatk_node}{featk\+Node}} attributes by assigning a unique id to each attribute and keeping records of the attribute order and name.
