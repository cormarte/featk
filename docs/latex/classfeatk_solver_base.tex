\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int Dimension, unsigned int Order$>$\newline
class featk\+Solver\+Base$<$ Dimension, Order $>$}

Base class for finite element solvers in Dimension dimensions. 

\mbox{\hyperlink{classfeatk_reaction_diffusion_solver}{featk\+Reaction\+Diffusion\+Solver}} is base class for finite element solvers in Dimension dimensions. \mbox{\hyperlink{classfeatk_reaction_diffusion_solver}{featk\+Reaction\+Diffusion\+Solver}} gathers \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}} and featk\+Boundary\+Condition objects involved in a finite element problem.

\mbox{\hyperlink{classfeatk_reaction_diffusion_solver}{featk\+Reaction\+Diffusion\+Solver}} provides functions for assembling global matrices and vectors from \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}} quantities getters as well as for applyingy essential boundary conditions to the global system matrix and vector.

\mbox{\hyperlink{classfeatk_reaction_diffusion_solver}{featk\+Reaction\+Diffusion\+Solver}} also defines the \mbox{\hyperlink{classfeatk_solver_base_a7fe7470913ee48379aded437886c9408}{update()}} routine as the sucession of calls to \mbox{\hyperlink{classfeatk_solver_base_a0041283d0bbb41b0955e511469ce3b65}{initialize()}} and \mbox{\hyperlink{classfeatk_solver_base_a616415cf09df676e1ba6bb139d54ea8c}{solve()}} functions. The \mbox{\hyperlink{classfeatk_solver_base_a0041283d0bbb41b0955e511469ce3b65}{initialize()}} function is used to initialize matrices and vectors that are not subject to change throughout the solving process. The \mbox{\hyperlink{classfeatk_solver_base_a616415cf09df676e1ba6bb139d54ea8c}{solve()}} function solves the proble√π itself, at once or iteratively. The \mbox{\hyperlink{classfeatk_solver_base_a1062d8f689866f75771e19349f27ae0a}{post\+Process()}} function is called at the end of the solve routine to assign solution and derivative quantities to the input \mbox{\hyperlink{classfeatk_mesh}{featk\+Mesh}}.

Derived classes must reimplement the \mbox{\hyperlink{classfeatk_solver_base_a616415cf09df676e1ba6bb139d54ea8c}{featk\+Solver\+Base\+::solve()}}, \mbox{\hyperlink{classfeatk_solver_base_acddf99bbfc1908f9a5e84c22a23e9f36}{featk\+Solver\+Base\+::get\+Global\+System\+Matrix()}}, and \mbox{\hyperlink{classfeatk_solver_base_a1062d8f689866f75771e19349f27ae0a}{featk\+Solver\+Base\+::post\+Process()}} functions and may reimplement the \mbox{\hyperlink{classfeatk_solver_base_a0041283d0bbb41b0955e511469ce3b65}{featk\+Solver\+Base\+::initialize()}} function if needed.
