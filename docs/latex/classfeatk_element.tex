\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int Dimension, unsigned int Nodes, unsigned int Boundaries, unsigned int Natural\+Dimension = Dimension$>$\newline
class featk\+Element$<$ Dimension, Nodes, Boundaries, Natural\+Dimension $>$}

Concrete implementation of a \mbox{\hyperlink{classfeatk_element_interface}{featk\+Element\+Interface}}. 

\mbox{\hyperlink{classfeatk_element}{featk\+Element}} is a the concrete implementation of an element with its attributes, nodes, shape function values and derivatives and integration rule. \mbox{\hyperlink{classfeatk_element}{featk\+Element}} provides routine member functions for numerical integration of usual quantities found in finite element problems.

\mbox{\hyperlink{classfeatk_element}{featk\+Element}} is templated over unsigned integers Dimension, Nodes, Boundaries and Natural\+Dimension (see definitions below). A given combination of these template parameters univoquely defines an element type. Member variables featk\+Element\+::integration\+Rule and featk\+Element\+::nodes\+Natural\+Coordinates and member functions \mbox{\hyperlink{classfeatk_element_a7f0ec045cecf67927669ab28bf244345}{featk\+Element\+::featk\+Element()}}, \mbox{\hyperlink{classfeatk_element_a843d80b2a0198a054dfbeadb2005a644}{featk\+Element\+::get\+Shape\+Function\+Natural\+Derivative\+Values()}} and \mbox{\hyperlink{classfeatk_element_adbdf8f2ee4f6ad9cb427a016e8eb0fb9}{featk\+Element\+::get\+Shape\+Function\+Values()}} have no default implementation and must be reimplemented for each template specialization.
