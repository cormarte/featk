Product Gaussian quadrature rule for numerical integration over a reference element. 

\mbox{\hyperlink{classfeatk_product_gaussian_quadrature_integration_rule}{featk\+Product\+Gaussian\+Quadrature\+Integration\+Rule}} implements product Gaussian quadrature rule for numerical integration over a reference element in natural coordinates. Integration points and weights are defined in Natural\+Dimension 1 and are computed using the product rule for Natural\+Dimension $>$ 1.

Constructor of \mbox{\hyperlink{classfeatk_product_gaussian_quadrature_integration_rule}{featk\+Product\+Gaussian\+Quadrature\+Integration\+Rule}} specializations initialize the inheritted member variable \mbox{\hyperlink{classfeatk_integration_rule_interface_a83d4c1ba06e35dd8f30efcfa0fba362b}{featk\+Integration\+Rule\+Interface\+::points\+And\+Weights}} according to their template parameter values.

See C. Felippa, Advanced Finite Element Methods\+: Chapter 11 -\/ The 8-\/Node Hexahedron, p. 10-\/11, 2017.

\begin{DoxyWarning}{Warning}
For a given Natural\+Dimension template parameter value, the Points template parameter value can only be 1, 2, 3, 4 or 5 to the power of Natural\+Dimension.
\end{DoxyWarning}
