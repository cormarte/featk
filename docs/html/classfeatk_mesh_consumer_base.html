Detailed Description

template<unsigned int Dimension>
class featkMeshConsumerBase< Dimension >

Base class for mesh consuming algorithms.

featkMeshConsumerBase is a base class for featkMesh consuming featkAlgorithm objects.

featkMeshConsumerBase stores featkMesh objects used as input by the featkAlgorithm.

Template Parameters
DimensionThe cartesian dimension of the algorithm.
