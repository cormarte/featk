Detailed Description

element node with its attributes including cartesian coordinates.

featkNode encapsulates element node attributes including its cartesian coordinates in a Dimension dimensional space.

Attributes are stored as Eigen::MatrixXd objects for ease but are accessed through Eigen::Matrix objects whose dimensions are fixed at compile time given the attribute order (0 for scalars, 1 for vectors, n for nth-order tensor). This ensures the adequacy of most Eigen::Matrix dimensions at compile time, limiting the risk of run time errors while benefiting from the template construction of Eigen.

featkNode object also store a pointer to each featkIElement it belongs to for computational efficiency.

Each node must be given a unique id at construction time whose uniqueness is ensured by the user.

Template Parameters
Dimensionthe number of cartesian dimensions of the node (1, 2, or 3).
