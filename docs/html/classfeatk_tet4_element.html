Detailed Description

Specialization of a 4-node tetrahedral featkElement.

featkTet4Element is a specialization of featkElement representing an isoparametric tetrahedron in Dimension 3 with 4 Nodes, 4 Boundaries parametrized in NaturalDimension 3. featkTet4Element provides implementations for the specialization of member variables featkElement::integrationRule and featkElement::nodesNaturalCoordinates and member functions featkElement::featkElement(), featkElement::getShapeFunctionNaturalDerivativeValues() and featkElement::getShapeFunctionValues().

featkTet4Element is parametrized as follows:
