Detailed Description

Mesh in Dimension dimensions with its elements and nodes.

featkMesh gather featkElement and featkNode objects defining the geometry of a finite element problem.

featkMesh manages featkElement and featkNode attributes by assigning a unique id to each attribute and keeping records of the attribute order and name.

Template Parameters
Thecartesian dimension of the mesh.
