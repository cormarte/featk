Detailed Description

template<unsigned int Dimension, unsigned int Order>
class featkBoundaryConditions< Dimension, Order >

Boundary conditions for finite element problems.

featkBoundaryConditions stores the ids of the fixed degrees of freedom of finite element problems along with their prescribed values.

For efficiency reasons during the assembly process (see featkSolverBase), distinction is made between zero prescrided values, for which the global system vector can be trivially modified, and non-zero prescrided values, involving iteration over non-zeros entries of the global system matrix.

Template Parameters
DimensionThe cartesian dimension of the problem.
OrderThe order of the variable the system is solved for.
