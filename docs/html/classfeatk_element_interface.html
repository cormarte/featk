Detailed Description

Interface for featkElement objects in Dimension cartesian dimensions.

featkElementInterface is an interface allowing to handle featkElement objects with Dimension cartesian dimensions and various Nodes, Boundaries and NaturalDimension template parameter values. featkElementInterface makes it possible to define hybrid featkMesh seamlessly.

Template Parameters
DimensionThe cartesian dimension of the element.
